sell项目中遇到的坑

父组件异步获取数据传递到子组件中

父组件通过axios异步获取数据，传递给子组件，发现子组件的created先于父组件的created执行，子组件获取不到数据；
解决方法：父组件template标签中引入子组件的时候添加一个判断条件，判断要传的数据是否为空，不为空时就渲染子组件，注意此时要传的数据初始值要设为undefined，不能设为空对象{}。

图标字体

官网：https://icomoon.io/#home

具体步骤：
1、进入官网，点击有上角 IcoMoon App，点击左上角的Import Icons，导入svg格式的文件；

2、选中需要生成图表文字的icons，点击右下角的 Generate Font，生成图表文字，并点击右下角Download进行下载；

3、下载的文件将其中的style.css、fonts文件夹复制到根目录下的static文件夹下，；

4、直接在根目录下的 index.html 中通过 引入style.css 或者在要用到的页面通过 import 引入

5、style导入需要的三个依赖、包（了解）：stylus-loader、postcss-loader、css-loader，通过终端下载即可：

6、在需要图标字体的地方加上style.css中相应图表的类名
