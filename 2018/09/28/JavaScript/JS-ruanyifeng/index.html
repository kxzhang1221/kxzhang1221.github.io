JavaScript特点

  • 操控浏览器的能力：唯一一种通用的浏览器的内置脚本语言；
  • 广泛的使用领域：JavaScript正在向通用的系统语言发展，HTML5、Node.js、NoSQL等；
  • 易学性：学习环境无处不在（浏览器）、简单性、与主流语言的相似性；
  • 强大的性能：灵活的语法，表达力强、支持编译运行、事件驱动和非阻塞式设计；
  • 开放性：统一的国际标准；
  • 社区支持：极大的社区、丰富的资源。

JavaScript的历史

1992年底，人类历史上第一台浏览器：Mosaic（美国国家超级电脑应用中心 NCSA开发）；

1995年12月，Netscaoe公司与Sun公司一起发布了JavaScript；

1997年7月，ECMA组织规定浏览器脚本语言的标准，并将这种语言成为ECMAScript；

ECMAScript和JavaScript的关系

  • ECMAScript式JavaScript的规范；
  • JavaScript是ECMAScript的一种实现。

语法概述

基本句法和变量

语句

JavaScript程序的执行单位为 行（line），每行叫做一个语句。

语句：是为了完成某种任务而进行的操作。

语句以分号结尾，多条语句可以写在一个行内。

表达式：一个为了得到返回值的计算式。

语句主要为了进行某种操作；表达式则是为了得到返回值。

变量

变量：是对值得引用。

使用变量等于引用一个值，每一个变量都有一个变量名。

变量声明使用关键字 var，且var声明的变量不能用detele命令删除。

单一var原则：用一个var声明多个变量；

JavaScript是一种动态类型语言，即变量的类型没有限制，可以赋予各种类型的值。

变量提升

JavaScript引擎的工作方式是：先解析代码，获取所有被声明的变量，然后再一行一行地运行。

变量提升：所有的变量的声明语言都会被提升到代码的头部。

标识符

标识符：用来识别具体对象的一个名称（严格区分大小写）。

标识符命名规则：

  1. 第一个字符可以是任意的Unicode字母，以及美元符号（$）和下划线（_）；
  2. 第二个字符和后面的字符，再第一点基础上还可以用数字；
  3. 中文是合法的标识符，可以用作变量名。

JavaScript保留字：
arguments、break、case、catch、class、const、continue、debugger、default、delete、do、else、enum、eval、export、extends、false、finally、for、function、if、implements、import、in、instanceof、interface、let、new、null、package、private、protected、public、return、static、super、switch、this、throw、true、try、typeof、var、void、while、with、yield。

还有三个词，不是保留字：Infinity、NaN、undefined。

注释

注释：源码中被JavaScript引擎忽略的部分，作用是对代码进行解释。

JavaScript提供两种注释：

  1. 单行注释：用 // 起头；
  2. 多行注释：放在 /* 和 */ 之间。

区块

区块：JavaScript使用大括号，将相关的语句组合在一起。

JavaScript的区块不构成单独的作用域。即区块中的变量就是全局变量。

条件语句

if结构
switch结构

switch结构：适用于判断条件存在较多情况的时候。每种case代码块内部的break语句不能少。

循环语句

while循环

while语句包括一个循环条件，只要该条件为真，就不断循环。

for循环

for语句分为三步：

  1. 确定初始值；
  2. 检查循环条件，只要为真就进行后续操作；
  3. 后续操作完成，返回上一步，检查循环条件。

格式如下：
for (初值; 循环条件; 下一步)

所有for循环，都可以改写成while循环

do…while循环

do…while 循环与while 循环类似，唯一的区别就是先运行一次循环体，然后判断循环条件。

break语句和continue语句

break语句：用于再循环体中跳出循环；

continue语句：用于不在进行本次循环的后续操作，直接进入下一次循环。

数据类型

原始类型和合成类型

JavaScript的值的类型分为两大类：原始类型和引用类型；

原始类型：数值（number）、字符串（string）、布尔值（boolean）
引用类型：对象（object）、数组（array）、函数（function）

对象和数组是两种不同的数据组合方式，函数其实是处理数据的方法。

JavaScript还定义了一个特殊的数据类型 undefined 和一个特殊的值 null；

  • undefined：表示未定义，即还没有确定的数据类型；（变量声明但没被赋值，则它默认为undefined）
  • null：表示一个空对象，不是一种单独的数据类型，而是包含在对象类型中的一个特殊值。

JavaScript的所有数据，都可以视为对象。

数据类型分类：

  • 值类型（基本类型）：数值（number）、字符串（string）、布尔值（boolean）、对空（null）、未定义（undefined）、Symbol（ES6新增）；
  • 引用数据类型：对象（object）、数组（array）、函数（function）

字符串

字符串：若干个排在一起的字符，放在单引号或双引号之中。

单引号、双引号交替使用，不然就只能用转移符 \ 进行转义。

如果一个字符串变量需要写成多行，可以在每行的结尾加上反斜杠 \。

JavaScript内部将字符串是为字符串组成的数组，方括号运算符遵循数据运算规则。

布尔值

布尔值：包含true和false两个值。

返回布尔值的运算符：

  • 两元逻辑运算符：&&(and)、||(or)
  • 前置逻辑运算符：!(not)
  • 相等运算符：===、!==、==、!=
  • 比较运算符：>、>=、<、<=

JavaScript预期某个位置应该是布尔值，则会将该位置上现有的值自动转为布尔值。

转换规则：

  • 转为false：undefined、null、false、0、””、NaN
  • 其他都转为 true

特别注意：空数组、空对象对应的布尔值，都是true

typeof运算符

typeof运算符：可以返回一个值的数据类型。

typeof运算符的运算结果：

  1. 数值、字符串、布尔值分别返回number、string、boolean。
  2. 函数返回function
  3. undefined返回undefined（可以用typeof检查一个没有声明的变量，不报错）
  4. 其他的都返回object

结尾的分号

不适用分号结尾的语句

  1. for和while循环
  2. 分支语句：if、switch、try
  3. 函数的声明语句

以上三种情况，如果使用了分号，并不会报错。解释引擎会把这个分号解释为空语句。

分号的自动添加

JavaScript引擎会在该写分号却没写的情况下，自动添加分号。

一般来说，在没有分号结尾的情况下，如果下一行起首的是 (、[ 、+、-、/这五个字符中的一个，分号不会被自动添加。只有下一行的开始与本行的结尾，无法放在一起解释，JavaScript 引擎才会自动添加分号；

另外，如果一行的起首是“自增”（++）或“自减”（–）运算符，则它们的前面会自动添加分号；

如果continue、break、return 和throw 这四个语句后面，直接跟换行符，则会自动添加分号。

数值

JavaScript 内部，所有数字都是以 64 位浮点数形式储存，即使整数也是如此。由于浮点数不是精确的值，所以涉及小数的比较和运算要特别小心。

数值的表示法

数值的表示法：字面形式、科学计数法。

以下两种情况自动将数值转为科学计数法：

  1. 小数点前的数字多于21位；
  2. 小数点后的0多于5个。

除去以上两种，其他的都采用字面形式直接表示。

NaN

NaN：表示非数字（not a number），主要出现在将字符串解析成数字出错的场合。

NaN不是一种独立的数据类型，而是一种特殊数值，它的数据类型依然属于Number。

NaN不等于任何值，包括它自身。NaN在布尔运算时被当作false。

isNaN：该方法可以用来判断一个值是否为NaN。

但是isNaN只对数值有效，传入其他值都会先被转为数值。

Infinity

Infinity：表示无穷，有正负之分，

Infinity 减去或除以Infinity，得到NaN。

parseInt方法

parseInt 方法可以将字符串或小数转化为整数。如果字符串头部有空格，空格会被自动去除。

如果字符串的第一个字符不能转化为数字（正负号除外），返回NaN。

parseInt 方法还可以接受第二个参数（2 到36 之间），表示被解析的值的进制。

parseFloat方法

parseFloat 方法用于将一个字符串转为浮点数。

如果第一个字符不能转化为浮点数，则返回NaN。

对象

对象：指的是一种数据结构，由若干个“键值对”（key-value）构成。键名与键值之间用冒号分隔。

但是如果键名不符合标识名的条件（即包含数字、字母、下划线以外的字符，且第一个字符不是数字），则必须加上引号。

对象生成方式

请注意，如果使用方括号运算符，键名必须放在引号里面，否则会被当作变量处理。

读取一个不存在的键，会返回undefined。可以用来检测一个变量是否被声明。

点运算符和方括号运算符，不仅可以用来读取值，还可以用来赋值。

查看一个对象本身的所有属性，可以使用Object.keys 方法。

属性的增加与删除

JavaScript 允许属性的后绑定，也就是说，你可以在任意时刻新增属性。

delete 命令可以删除属性，如果删除成功，返回布尔值true。

对象的引用

如果不同的变量名指向同一个对象，那么它们都是这个对象的引用，也就是说指向同一个内存地址。修改其中一个变量，会影响到其他所有变量。

in运算符

in 运算符用于检查对象是否包含某个属性名，如果包含就返回true。

在JavaScript 语言中，所有变量都是它的上层对象的属性（顶层对象除外），因此可以用in 运算符判断一个变量是否存在。

类似数组的对象

类数组对象：看上去像数组，有length属性，但无法使用一些数组方法。

类数组对象包括：arguments、大多数DOM元素集。

数组

数组：是按次序排列的一组值，数组的本质也属于对象。

定义

数组和对象的区别

  1. 对象的每个值都有对应的键名；而数组不需要指定键名，键名默认为0，1，2…；
  2. 对象成员引用可以使用点结构和方括号；数组只能使用方括号；

检查某个键是否存在的运算符 in，适用于对象，也适用于数组。

数组遍历：

  1. 使用for in循环
  2. for循环结合length属性。

数组的创建：

  1. 方括号创建；
  2. JavaScript内置Array构造函数创建，new Array()；

length属性

length属性：表示数组的元素个数。

length的特性：

  1. 添加新元素后，length会自动增长；
  2. length属性是可写的；length变小，数组的元素会自动减少到length个数；
  3. 清空数组的方法，length属性设为0；
  4. 设置length大于当前元素个数，数组会自动增长到该length值，新增位置填入空元素；
  5. length为不合法（负数或者大于2的32次方）的值，会报错；
  6. 使用delete删除一个值，只会把当前位置置为undefined，不会影响length属性的值；
  7. 为数组添加属性（非数字属性），不影响length属性的值。

数组的空位

空位：数组的某个位置是空元素（比如两个逗号之间没有任何值、delete命令删除一个值）。

空位会被计入length属性，值为undefined。

注意：使用length属性遍历，空位不会被跳过；使用数组的forEach方法或for in结构进行遍历，空位就会被跳过。
解决方法：在空位显示的填入undefined。

数组的操作

push(ele)：添加元素到数组末尾；
pop()：删除数组末尾的元素；
shift()：删除数组最前面（头部）的元素；
unshift(ele)：添加元素到数组的头部；
splice(index, n)：通过索引删除元素；
slice()：复制一个数组。array.slice()

ES5新增的数组方法

ES5 在数组原型（Array.prototype）上，新增了9 个方法，其中有7个与函数式（functional）操作有关：map、forEach、filter、every、some、reduce、reduceRight。

这7个方法可以直接在数组上使用，它们的参数是一个函数。这个作为参数的函数本身又接受三个参数，分别是数组的 当前元素elem、该元素的位置index 和 整个数组arr。

map

map：对所有元素依次调用一个函数，根据函数结果返回一个新函数。

forEach

forEach：对所有元素依次遍历，执行一个函数，它与map的区别在于不返回新数组。

filter

filter：对所有元素调用一个测试函数，操作结果为true 的元素组成一个新数组返回。

filter 方法接受一个 返回布尔值 的测试函数作为参数。该测试函数的第一个参数是当前数组成员的值。除了这个参数以外，其他参数都是可选的，分别是当前数组成员的位置和整个数组。

some

some：对所有元素调用一个测试函数，只要有一个元素通过该测试，就返回true，否则返回false。

every

every：对所有元素调用一个测试函数，只有所有元素通过该测试，才返回true，否则返回false。

reduce和reduceRight

reduce：从左向右依次处理数组的每个元素，最终累计为一个值。
reduceRight：从右向左依次处理数组的每个元素，最终累计为一个值。

reduce 方法的第一个参数是一个处理函数。该函数接受四个参数，分别是：

  1. 用来累计的变量（即当前状态）– 必须，默认为0；
  2. 数组的当前元素 – 必须；
  3. 当前元素在数组中的序号（从0开始）；
  4. 原素组。

要对累计变量指定初值，可以把它放在reduce方法的第二个参数。

indexOf：返回给定元素在数组中第一次出现的位置（index），如果没有出现则返回-1。
lastIndexOf：返回给定元素在数组中最后一次出现的位置，如果没有出现则返回-1。
