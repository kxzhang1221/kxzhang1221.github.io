函数

编程原则：高类聚，弱偶合；（偶合：重复代码）

函数是另一种类型的变量；

函数名，变量名：小驼峰 命名规则；

函数定义

函数声明

命名函数表达式

function abc(){} 是一个表达式，表达式是没有名字的；

匿名函数表达式 — 函数表达式(常用)

实参列表arguments

arguments 属性代表传入函数的实参，它是一个类数组对象。

由于JavaScript 允许函数有不定数目的参数，所以我们需要一种机制，可以在函数体内部读取所有参数。这就是arguments 对象的由来。

arguments中的值和形参关系：对应的值一个变另一个也变(映射关系)，但不是同一个人

递归

递归思路：先return，再找出口写在return之前；
1、找规律(公式) – return 公式;
2、找出口(已知条件)

递归：唯一好处就是代码简洁；

圆括号运算符和return语句

圆括号运算符：调用函数时，要使用圆括号运算符；圆括号之中可以加入函数的参数。
return语句：表示返回；JavaScript引擎遇到return语句，直接返回return后面的表达式的值，不再执行后续的任何语句。

第一等公民

JavaScript的函数与其他数据类型处于同等地位，可以使用其他数据类型的地方就能使用函数。

比如，

  • 可以把函数赋值给变量和对象的属性；
  • 把函数当成参数传入其他函数；
  • 把函数作为函数的结果返回（闭包）。

作用域

作用域：变量（变量作用域又称上下文）和函数生效（能被访问）的区域；

Javascript 只有两种作用域：

  • 一种是全局作用域，变量在整个程序中一直存在；
  • 另一种是函数作用域，变量只在函数内部存在。

预编译

预编译：函数声明整体提升，变量只有声明提升（有赋值的话，赋值不提升）

  • imply global 暗示全局变量

任何变量，如果变量未经声明就赋值（没有经过var声明），此变量就为全局对象所有；（全局对象：window）

  • 一切声明的全局变量，全是window的属性;

等价于

此时：window.b = 123; window.a = undefined

函数预编译四部曲

预编译发生在函数执行的前一刻

  1. 创建AO对象
  2. 找形参和变量声明，将变量和形参名作为AO属性名，值为undefined
  3. 将实参值和形参统一
  4. 在函数体里面找函数声明，值赋予函数体
实例解析

  1. 创建GO对象(Global Object) GO === window

  3. 在函数体里面找函数声明，值赋予函数体

闭包

闭包：但凡是内部的函数被保存到外部，一定形成闭包；

危害：闭包会导致作用域链不释放，造成内存空间泄露。
内存泄露：内存被占用，可用的资源就变少了，就相等于内存泄漏；

闭包的作用

闭包的作用：

  • 实现共有变量；
  • 缓存(存储结构)；
  • 实现封装函数使变量私有化；
  • 模块化开发；
