数据与方法

当一个 Vue 实例被创建时，它向 Vue 的响应式系统中加入了其 data 对象中能找到的所有的属性。
当这些属性的值发生改变时，视图将会产生“响应”，即匹配更新为新的值。

当这些数据改变时，视图会进行重渲染。

实例生命周期钩子

每一个Vue实例在创建时都需要初始化，比如设置数据监听、编译模板、将实例挂载到DOM并在数据变化时更新DOM等；
这个过程也会运行一些叫生命周期钩子的函数，给了用户在不同阶段添加代码的机会。
钩子在实例生命周期的不同阶段被调用，如created、mounted、updated和destroyed；

Vue.js简介

Vue.js：一套构建用户界面的渐进式框架；简单小巧的核心，渐进式技术栈，足以应付任何规模的应用。

Vue关注点：视图层，采用自底向上增量开发的设计；
Vue的目标：通过尽可能简单的API实现响应的数据绑定和组合的视图组件；

Vue.js 提供了现代Web开发中常见的高级功能，比如

  • 解耦视图与数据
  • 可复用的组件
  • 前端路由
  • 状态管理
  • 虚拟DOM

MVVM模式

Vue.js同Angular、Ember一样采用 MVVM（Model-View-View-Model） 模式。

MVVM模式分为Model、View、ViewModel三块，View和ViewModel之间通过双向数据绑定（data-binding）建立联系。

Vue.js的不同

jQuery，操作DOM、绑定事件等原生JavaScript能力，功能复杂不便于维护；
Vue.js通过MVVM模式拆分为 视图 与 数据 两部分，将其分离。我们只需要关注数据即可，DOM操作Vue会自动搞定。
